Herança e Especificidade


Em Herança tratamos sobre atributos gerais que incluimos em elementos, na qual todos os elementos contidos dentro desse elemento "pai" acabam recebendo esses atributos automaticamente, e posteiormente podemos incluir atributos mais especificos caso quisermos.


Em Especificidade teremos que para dois seletores iguais, o ultimo atributo no código é que será aplicado aos elementos. [ Salvo quando colocado o atributo "!important" para algum dos atributos, ai o navegador vair entender que aquela formatação é mais importante que as outras - porém não é recomendado que seja atilizado com frequencia ].

Os Seletores mais especificos também terão os atributos mais importantes para o navegador aplicar nos elementos.

Menu Vertical

Nesta seção da barra lateral tivemos os insights abaixo:


- Para Listas, sempre colocar o comando "list-style-type: none" no Seletor "ul" para limpar a formatação padrão das listas.


- Reposicionamento de uma imagem ao passar o mouse [ usando os comandos = "background-position: right bottom" no estado HOVER do link + "background: url() no-repeat;" no BACKGROUND do Seletor "a" + um Background #f7b600 que é parecido com o da imagem para caso a imagem selecionada não seja carregada.


- Usamos o comando "line-height" para centralizar o text verticalmente dentro do Seletor "a".


- Para alguns itens é necessário atribuir "display: block;" para que o mesmo receba certos atributos ( que caso seja um elemento in-line, não receberia ).

Layout de Largura Fixa e Layout Líquido


Este tópico trata sobre ajustar os elementos automaticamente conforme realizamos o redimencionamento da janela do navegaor. Neste arquivo de estudos á está sendo utilizado essas práticas de redimencionamento [ Lembrando que isso não chegamos no módulo de Bootstrap, no qual teremos um estudo especifico sobre Layouts para dispositivos Desktop e Mobile - Responsívos ].


OBS.: Por questão de usabilidade, podemos colocar a DIV conteúdo, antes da DIV navegação. Para que leitores de tela, para pessoas que não enxergam, possam ler primeiro o conteúdo da página, e logo após isso, ler a barra de navegação.


- Para travar a largura mínima de uma tela usamos o atributo = "min-width"

- Para travar a largura máxima de uma tela usamos o atributo = "max-width"

- Para travar a altura mínima de uma tela usamos o atributo = "min-height"

- Para travar a altura máxima de uma tela usamos o atributo = "max-height"


OBS.: Para realizar o ajuste de imagens, é da mesma forma de reajustar as DIVs
